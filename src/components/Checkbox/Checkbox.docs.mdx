# Checkboxes

Use checkboxes when the user can select more than one option from a list. Make clear with helper text that this is the case. Use [radio buttons](?path=/story/components-radio--default-radio) when the user can choose only one option from a list.

Source: [https://cfpb.github.io/design-system/components/checkboxes](https://cfpb.github.io/design-system/components/checkboxes)
